# 概述

    今天看文章看到有篇介绍随机的本质，看完之后发现对贝叶斯的理解有帮助，就写篇文章记录下，到时候再次去看贝叶斯概率的时候，应该会有帮助。

# 抛骰子例子

    想象一颗骰子,如果各面材质是均匀的,那么每个面朝上的概率都是 1/6 .但是,如果我们知道抛出骰子的初始速度和角度,以及风速、空气湿度、地面材质、骰子材质等信息,理论上可以用计算机模拟哪个面会朝上.我们对这些信息了解得越充分,预测也就越准确.如果我们掌握了影响骰子哪个面朝上的全部信息,那么骰子哪个面朝上对我们来说就不再是不确定事件,而变成了一个确定事件!

    因此随机性来源于信息的缺失,也即无知.了解的信息越多,对一事件发生的概率最终会趋向于 1或者 0;当我们掌握了全部信息后,不确定性事件就会变成了确定性事件.

    上述直觉可以用现代概率论严格证明,本质上是Doob鞅收敛定理的特例.

# 某人家房子下埋着石油的概率是多少?

    为了更好地理解"概率来源于无知",再看一个问题:某人家房子下埋着石油的概率是多少?

    客观上,房子下要么有石油,要么没有,这是一个完全确定的事件.但是在勘探之前,我们无法得知地下是否埋着石油,此时对我们而言这就是一个概率事件.给定某地区,我们甚至能准确估计这个概率.此类"本质并非随机,但由于无知而显得随机"的现象,有些人也称为"效果随机".效果随机具有很强的主观性,它取决于个人的无知程度.仍以石油为例:如果小张勘探后发现没有石油,对于他来说这就不再是随机性问题,而变成确定性问题了.而房主在不知情时仍会认为有概率埋着石油.但是如果某天小张把勘测结果告诉房主,房主的判断也会瞬间从不确定转为确定.

    一般而言,越详尽地搜集信息和分析,判断也就越准确;但信息的搜集和保存也有成本,需要适可而止.

    我们在日常生活中遇到的几乎所有``随机事件''都是效果随机.也难怪拉普拉斯感叹:如果有一个"神"知道宇宙某一时刻所有粒子的初始条件,它就能推演过去和未来;在它眼中,一切皆注定,随机性无处容身.这是机械论与决定论的世界观,包括爱因斯坦在内的众多科学家都笃信这一点.

    然而,量子力学带来了冲击.它认为微观粒子在观测前并没有一个明确的位置,只有当我们观测时才会坍缩到某处,而具体坍缩到哪里是完全随机的,我们只能描述其概率密度.也就是说,量子力学的世界观主张宇宙中存在"真随机"!

# 《狐狸与刺猬:专家的政治判断》

    狐狸与刺猬的典故来自古希腊寓言:狐狸懂得很多小事,刺猬只知道一件大事.泰特罗克发现,"狐狸型"专家的预测准确率明显高于"刺猬型"专家.

    狐狸型"专家,是指那些涉猎广泛,能够综合利用各种信息作出判断的人;"刺猬型"专家也许在单一领域很强,但是对于自己专业以外的知识所知甚少,因此难以跳出自我的框架.真实世界是非常复杂的,单一学科只提供了现实世界的局部投影,"刺猬型"专家往往陷入片面的信息,作出的判断也就不可能准确.而"狐狸型"专家善于整合不同学科的知识和信息,因而他们的"条件概率"中所依赖的"条件"也更充分,作出的预测自然更可靠.这再次印证了前面讲过的道理:从概率角度来看,对信息的把握程度越高,对概率的预测也就越准确.

    讽刺的是,往往"刺猬型"专家对自己的预测相当自信,他们在频频在媒体上断言下一次金融危机的时间,或气候变暖带来的生态危机和环境破坏,但是他们的预测常常错得离谱.反而是"狐狸型"专家,对自己的预测非常谨慎,因为他们很清楚自己不知道的信息还有很多.随着新信息的不断涌入,"狐狸型"专家会积极调整自己的预测并承认自己的错误,事实上,这正是贝叶斯方法的核心精髓.
