# 概述

    图是一个很大的概念，也是一个比较复杂的数据结构及基础学科。在计算机这块，我涉及到的主要是图算法，图可视化，图计算，图存储等。

# 图可视化

    图可视化分为3种类型: 1. 通用的可视化工具;2. 图可视化框架;3. 基于图数据库的可视化平台

## 通用的可视化工具
    
    通用的可视化工具不需要依赖图数据库作为底层数据源，他们的依赖可以是文件，关系数据库等。
    常见工具有: 
        1. ClueMaker
        2. Gephi
        3. Linkurious
        4. VOSViewer
        5. puppygraph

## 图可视化框架

    图可视化框架由多个组件组成：
        1. 数据处理组件
        2. 可视化组件
        3. 交互组件
        4. 布局组件
    
    常见工具有:
        1. GalaxyVis
        2. D3.js
        3. cytoscape
