# 七种语义介绍

  我将把广义上的“语义”分解为七个不同的组成部分。
  
  1. 概念语义（或我更倾向于称之为逻辑语义），

  2. 内涵语义、

  3. 社会语义、
  
  4. 情感语义、
    
  5. 反映语义、
    
  6. 搭配语义、
     
  7. 主题语义。

## 概念意义

  它具有复杂而精妙的组织结构，这种结构可以与语言句法和音系层面的类似组织结构相比较，并相互关联。特别是，我想指出两个似乎是所有语言模式基础的结构原则：对比原则和结构原则。

  1. 对比原则

  2.结构原则
    是指较大的语言单位由较小的单位构成；或者（从相反的角度来看），是指我们能够从句法角度分析一个句子，将其分解成各个组成部分，从其直接成分开始，通过层级划分，最终到达其最根本的成分或最小的句法元素。

  对于任何句子的分析，我们需要建立“语音表征”、“句法表征”和“语义表征”，以及一个表征层级如何，从另一个表征层级推导出来。

  概念语义学的目标是，对于任何给定的句子解释，提供一个由抽象符号构成的配置，即其“语义表征”，它准确地展示了我们需要知道的信息，以便将该含义与语言中所有其他可能的句子含义区分开来，并将该含义与正确的句法和语音表达相匹配。
  层级匹配的能力是单向的，如果我们是在解码，即听一个句子并解释它；

  如果我们是在解码，即听一个句子并解释它；

  如果我们是在编码，即编写和说出一个句子，则层级匹配的能力是反向的（C-*B-*A）。由此可见，概念意义是语言不可分割且至关重要的组成部分，以至于我们几乎无法在不提及概念意义的情况下定义语言。

  一种语言，如果其交流方式并非通过概念意义，
    例如，一种“语言”，
    仅仅通过诸如“哦！啊！哦吼！
    唉！”和“加油！”之类的感叹词进行交流，那么它就根本不能称之为语言，
    因为我们通常用“语言”一词来指代人类的语言。

## 内涵语义

  当我们将其与内涵意义进行对比时，概念意义的更多独特之处就会显现出来。内涵意义是指一个表达方式因其所指对象而具有的交际价值，超越了其纯粹的概念内容。在很大程度上，指称的概念与概念意义有所重叠。如果“女人”一词的概念定义包含三个特征（+人类，-男性，

+成年），那么“人类”、“成年”和“女性”这三个属性就必须构成正确使用该词的标准。这些对比特征，用“现实世界”的术语来表达，就成为了指称对象（该词所指的对象）的属性。但是，我们还习得了许多额外的、非标准性的属性，这些属性也被我们视为“女人”指称对象所应具备的。这些特征不仅包括生理特征（例如“有子宫”），还包括心理和社会属性（例如“合群”、“受母性本能支配”），甚至可能延伸到一些仅仅是女性特有的而非不变的特征（例如“会说话”、“擅长烹饪”、“穿裙子或连衣裙”）。此外，内涵意义还可以包含“假定的”

属性，这取决于个人、群体或整个社会所采取的视角。因此，过去女性被赋予了诸如“脆弱”、“易哭”、“胆小”、“情绪化”、“不理智”、“善变”等特质，这些特质是占主导地位的男性乐于强加给她们的，同时，女性也被赋予了诸如“温柔”、“富有同情心”、“敏感”、“勤劳”等更合适的品质。显然，词语的含义会随着时代和社会而变化。

一百年前，“不穿裤子”似乎成了“女人”一词及其在欧洲语言中的对应词的绝对定义，就像今天在许多非西方社会一样，女性也被赋予了与我们自身思维方式截然不同的特质。同样显而易见的是，即使在同一语言群体中，词语的含义也会因人而异。对于一个说英语的厌女者来说，“女人”一词会带有许多贬义的联想，而这些联想在持更女权主义观点的人心中并不存在。

  很显然，在谈论内涵时，我实际上指的是人们在使用或听到某个表达时所联想到的“现实世界”体验。因此，概念意义和内涵意义之间的界限与“语言”与“现实世界”之间的区别相吻合。这解释了为什么人们会觉得内涵在某种程度上是语言的附带部分，而不是其本质组成部分。我们也可以注意到，内涵意义并非语言所独有，而是其他交流系统（例如视觉艺术和音乐）所共享的，这印证了上述观点。
“婴儿”一词的任何内涵都可以通过一幅婴儿的图画（更有效地，因为媒介是直接具象的）唤起，或者模仿婴儿的哭声。语言内涵和视觉内涵之间的重叠在广告中尤为明显，在广告中，文字往往是次要的。在赋予产品光环效应（即正面联想）的任务中，插图的作用

  第二个表明内涵意义相对于概念意义而言是次要的事实是，内涵意义相对不稳定：也就是说，正如我们所见，它们会根据文化、历史时期和个人经验而发生显著变化。虽然认为所有使用特定语言的人都说着完全相同的语言过于简单，但可以假设，作为一项基本原则（没有这项原则，通过该语言进行交流是不可能的），
他们总体上共享相同的概念框架，就像他们共享大致相同的句法一样。事实上，一些近期的语义学家假设，相同的基本概念框架是所有语言共有的，并且是人类思维的普遍属性（参见第26-30页）。

  第三，内涵意义在某种意义上是不确定的和开放的，而概念意义则不是。内涵意义的开放方式与概念意义相同。由于我们对宇宙的认知和信念是开放的：指称对象的任何特征，无论是主观识别的还是客观识别的，都可能影响指称该对象的表达的内涵意义。
与之相反，语义理论通常认为，词语或句子的概念意义可以用有限的符号集来编码（例如，以有限的离散意义特征集的形式），并且句子的语义表征可以用有限数量的规则来指定。这种关于概念内容的有限性和确定性的假设，是基于语言学家在分析语言结构的其他方面时通常所做的假设而建立的。
  
  这些假设在某种程度上过于简化，但如果没有它们，就很难坚持语言是一个有限且连贯系统的观点。

## 社会语义和情感语义

  现在我们转向与话语发生的语境相关的两个沟通方面。

  社会意义是指一段语言所传达的关于其使用的社会环境的信息。在某种程度上，我们通过识别同一语言中不同的风格维度和层次来“解码”文本的社会意义。
  我们识别出一些词语或发音是方言，这告诉我们一些关于说话者地理或社会出身的信息；
  语言的其他特征告诉我们一些关于说话者和听话者之间社会关系的信息：
  我们有一个“地位”使用的等级，从一端的正式和文学英语，到另一端的口语、亲切，最终到俚语英语。
  
  一项研究（Crystal 和 Davy，《英语风格研究》）
  识别出了以下社会风格变异的维度（我添加了一些在每个维度上区分的使用类别示例）：
  变异依据：
    方言（某个地理区域或社会阶层的语言）
    时间（十八世纪的语言等）
    省份（法律语言、科学语言等） （例如广告等）
    语态（礼貌用语、口语、俚语等）
    语态（备忘录、讲座、笑话等的语言）
    独特性（狄更斯、海明威等的风格）

  虽然并不详尽，但这份清单表明了单一语言中可能存在的风格差异范围。

  因此，我们很少找到概念意义和风格意义都相同的词语，这并不奇怪。这一观察结果常常导致人们宣称“真正的同义词并不存在”⁵。如果我们把同义词理解为交际效果的完全等价，那么确实很难找到反驳这一论断的例子。
  但是将“同义词”一词限定为概念意义的等价性会方便很多，这样我们就可以对比概念上的同义词，并考察它们不同的风格语调：

  在区分同义词时，“地位”这一风格维度尤为重要。以下示例展示了地位差异如何贯穿整个句子，并体现在句法和词汇上：
    (1) 他们朝警察扔了一块石头，然后带着赃物逃之夭夭。
    (2) 朝警察扔了一块石头后，他们带着钱逃走了。
  句子 (1) 可能是两个罪犯事后随意谈论犯罪时说的；句子 (2) 可能是总督察在撰写正式报告时说的。两者描述的都是同一件事，它们概念意义的共同点在于，任何人都很难认同其中一个句子的真实性，同时否定另一个句子的真实性。

  从更具体的意义上讲，社会意义可以包括所谓的言语行为力
    例如，它是否应被解读为请求、断言、道歉、威胁等等。言语在此方面所发挥的功能可能与其概念意义只有间接的关联。句子“我没有刀”在形式和意义上都是一个陈述句，但在社交现实中（例如，对餐厅服务员说），它很容易变成一个请求，例如“请给    我一把刀”。由此，我们只需稍稍思考一下语言是如何反映说话者的个人感受的，包括他对听者的态度，或者他对所谈论事物的态度。

  这种意义可以称为情感意义，它通常通过所用词语的概念或内涵内容来明确传达。
    例如，有人对某人说：“你是个恶毒的暴君，是个卑鄙的恶棍，我恨你！”对方几乎不会对说话者对他的感受产生任何疑问。但表达态度的方式并非只有这一种，
    例如，通过调整我们的言辞，来体现我们的态度。为了让别人安静下来，我们可能会说：
      (3) 非常抱歉打断你们，但请问你们能否稍微降低一下音量？
      (4) 请你们提高音量。

  语调和音色等因素——我们通常所说的“语气”——在这里也很重要。(3) 中礼貌的印象可能会被尖刻的讽刺语气所颠覆；如果用温和请求的语气说出句子 (4)，它就可以变成亲密朋友之间的玩笑。
  情感意义在很大程度上是一种寄生性范畴，因为为了表达我们的情感，我们需要依赖其他意义范畴——概念意义、内涵意义或风格意义——的中介。通过风格表达情感的方式，
    例如，当我们采用不礼貌的语气来表达不满时（如上文(4)所示），或者当我们采用随意的语气来表达友好时。
    另一方面，语言中存在一些要素（主要是插入语，例如“啊哈！”和“耶！”），
    它们的主要功能就是表达情感。当我们使用这些要素时，我们无需任何其他语义功能的中介，就能传达感受和态度。
