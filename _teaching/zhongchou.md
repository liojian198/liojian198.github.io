# Kickstarter

  它不是一个在线商店，而是一个帮助艺术家、设计师、开发者和各类“创客”通过大众的零散资金，将创意变成现实的平台。

## 1. 它如何运作？

  Kickstarter 采用的是一种**“全有或全无”（All-or-Nothing）**的模式：
  
  发起者（Creators）：设定一个筹款目标和截止日期。只有在截止日期前筹集到了100%（或更多）的资金，发起者才能拿到钱。
  
  支持者（Backers）：如果你看中一个项目，可以承诺捐赠一定的金额（Pledge）。
  
  结果：如果筹款失败，支持者不会被扣费；如果筹款成功，发起者拿到资金开始生产，支持者通常会获得相应的回报（Rewards）。

## 2. 常见的项目类型

  Kickstarter 专注于“创意领域”，常见的类别包括：
  
  科技与硬件：如智能手表（Pebble 最初就是在这里成名的）、新型咖啡机、无人机。
  
  游戏：包括独立电子游戏和大量的桌游（目前桌游是该平台非常火爆的类别）。
  
  艺术与影视：独立电影、纪录片、画册。
  
  设计与出版：创意书籍、极简主义背包、多功能工具。

## 3. 核心特点

  不是商店：你“购买”的不是现货，而是一个承诺。项目可能会延期，甚至在极少数情况下失败（无法交付）。
  
  回报机制：支持者获得的通常是该产品的第一批实物、限量版周边，或者是与创作团队互动的机会。
  
  门槛与费用：任何人都可以提交项目，但必须符合 Kickstarter 的规则。如果众筹成功，平台会抽取约 5% 的服务费。

# Indiegogo：科技与创新的首选

  它是 Kickstarter 最大的竞争对手，主要区别在于其灵活度。
  
  特色：提供“灵活资助”模式（Flexible Funding）。即便没有达到 100% 的筹款目标，发起者也可以选择保留已筹集到的资金（Kickstarter 必须达标才能拿钱）。
  
  适合对象：科技创业者、硬件开发者、寻找首批种子用户的早期项目。
  
  优势：支持全球更多国家，且审核流程通常比 Kickstarter 更快。

# Indiegogo / BackerKit / Gamefound：桌游与垂直领域

  如果你是桌游、游戏或重度垂直领域的爱好者或开发者，这些平台可能比 Kickstarter 更专业。
  
  Gamefound：目前全球最专业的桌游众筹平台。它集成了“众筹+后期管理（Pledge Manager）”，让支持者可以更方便地在筹款结束后追加购买或管理运费。
  
  Crowd Supply：专注于开源硬件和极客电子产品。它对项目的筛选非常严格，但一旦入选，平台会提供深度支持（包括物流和销售），信誉度极高。

# Patreon：长期订阅与创作支持

  它和 Kickstarter 的“一锤子买卖”不同，走的是长线支持路线。
  
  特色：支持者（Patrons）每月支付固定金额（如 $5/月），以换取创作者的持续更新、独家内容或互动。
  
  适合对象：YouTube 博主、播客（Podcaster）、插画师、作家。
  
  优势：为创作者提供稳定的月收入，而不是一次性的大额融资。

# GoFundMe：个人求助与公益慈善

  这是全球最大的个人筹款平台，不强调“回报”，而强调“帮助”。
  
  特色：通常用于医疗急救、学费资助、社区公益或灾后重建。
  
  区别：支持者不求产品回报，纯粹是捐赠。它不强制要求设定一个截止日期。

# WeFunder / StartEngine：股权众筹（投资）

  如果你想通过众筹真正拥有一家公司的股份，而不是只拿一个产品。
  
  特色：这属于“股权众筹”。支持者不再是“Backer”，而是“Investor（投资者）”。
  
  适合对象：初创公司。普通人可以用几百美金投资他们看好的初创企业，如果公司将来上市或被收购，你将获得分红或收益。

# 国内众筹平台

## 文化创意与兴趣众筹（类似 Kickstarter/Gamefound）

### 摩点 (Modian)：

  地位：中国文化创意众筹的头部平台。
  
  核心领域：桌游（国内最大的桌游众筹地）、独立游戏、动漫周边、博物馆文创（如故宫文创）。
  
  特色：社区属性极强，粉丝参与度高，适合具备 IP 属性或创意设计类的项目。

### 造点新货（原淘宝众筹）：

  地位：阿里旗下的新品众筹平台。
  
  核心领域：IP 联名产品、数码周边、家居创意件。
  
  特色：背靠淘宝巨大的流量，众筹成功的项目可以直接转化为淘宝店铺商品。

### Bilibili 会员购（众筹频道）：

  特色：精准针对二次元用户，主要是动画衍生品、手办、虚拟主播周边。

## 科技新品与智能硬件（大厂生态链）

  由于京东众筹和苏宁众筹已先后停止运营，国内科技类的众筹目前高度集中在品牌自有的“新品首发”频道，本质上更接近于深度预售。

### 小米有品众筹：

  地位：目前国内科技硬件众筹最成功的平台。
  
  特色：严格的审核机制。小米通过众筹来测试市场对新品（如米家智能新品、第三方酷玩产品）的接受度。

### 华为商城（众测/众筹）：

  特色：主要针对华为 HarmonyOS 智联生态的第三方硬件
