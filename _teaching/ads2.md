# 区分流量本质

  你吸引的是 Attention “注意力” 还是Intention “购买欲望” ？销量上不去真的只是流量不够吗？

  做内容、做投放、做增长，最容易死在一个错觉里： “只要有流量，后面总能转化。”

  流量性质其实至少分成两种：一种只是路过，一种带着问题来找答案。

  先给结论：增长的第一步不是优化账户，是先把流量分清楚。你分不清，后面的“加码/止损/优化”都只会变得更混乱，严重妨碍后续归因和数据分析，进而影响长久的销量情况。

  同样 1 万流量，可能是两种完全不同的东西，他们的转化率也有极大的不同：
  
  1. 1 万个随手划过的注意力
  2. 1 万个正在找答案的明确需求

  |你拿到的用户当下的状态|你应该推的下一步动作|你更应该看的指标|
  |:--|:--:|--:|
  |注意力（Attention）：路过，随手看看|关注 / 收藏 / 进站 / 进入可复触达池|有效停留、收藏/转发、品牌词/站内检索、再营销成本|
  |意图（Intent）：带着问题来找答案|对比 / 咨询 / 加购 / 下单|CVR、CAC、LTV/CAC、复购|

  注意力(Attention) 的本质，是“低承诺互动”；
  
  意图（Intention）本质，是“高承诺选择”。Attention 不是购买力，它只是“进入你视野的许可”。
  
  你拿到的只是目光，还不是爱的号码牌。

  不区分这两种流量（注意力 vs 意图），你会在三个关键决策上难以抉择：
  
  1. 该加码的时候犹豫：其实来的是“意图流量”，已经接近成交了，你却因为报表数据不够漂亮不敢放量。
  2. 该止损的时候自我感动：其实来的是“注意力流量”，短期内很难转化，你却拿它当成交流量去优化，越投越不甘心。
  3. 预算越花越多，链路越跑越乱：你为了把“注意力流量”硬改成“能成交的流量”，不断改素材、改落地页、改事件，最后系统学不到、团队也复盘不出正确的因果。

  最后没有人能确定到底是哪一步出了问题。很多时候销量上不去不是缺流量，而是缺“流量识别能力”。这一步错了，后续的努力就都只是自我感动。


## 先分清：你拿到的是“注意力”，还是“意图”？

  别再说“我要流量”。先回答一个问题： 你现阶段要的是 Attention 注意力，还是 Intent 意图？

### A. 注意力流量（Attention）

  1. 注意力流量来自推荐系统。
  2. 用户并不在找你，但平台把你塞到用户眼前。典
  3. 型场景：信息流、短视频、图文推荐、KOL/KOC。
  4. 注意力流量真正能带走的，不是订单，而是这三件事：
     1)让用户承认“问题存在”
     2)让用户记住“你是谁”
     3)让用户未来在某个时刻想起你

  这部分流量买的其实是注意力，而不是购买力。

### B. 意图流量（Intent）

  1.意图流量来自匹配系统。
  2.用户已经在找方案，缺的是“更可信的选择”。
  3.典型场景：搜索、测评、对比页、品牌词。
  4.意图流量要解决的也很直接：
    1)把犹豫的人推过临界点
    2)把有需求的人拉进网站了解产品
    3)把准备下单的人留在你这里

  意图流量不是你自己找来的造来的，是已存在的需求直接承接出来的。
  你买的到底是什么流量，决定了你该写什么内容，怎么做转化，以及用什么指标来判断对错。

  注意力路径： 用户刷信息流---->推荐系统分发---->内容出现眼前---->关注/收藏/进站

  意图路径： 用户搜索关键词--->匹配系统承接--->你的内容/落地页---->对比/咨询/下单

## 再分一层：冷 / 暖 / 热，温度层决定打法

  同样是流量，也有温度差。在哪个温度层，决定了你该说哪句话。

  |温度|用户脑子里那句话|你最该补的“缺口”|典型内容形态|
  |:--|:--:|:--:|--:|
  |冷|“你凭什么？”|解释：把问题讲清楚，把机制讲通|观点/机制拆解/痛点命名|
  |暖|“我怕买错。”|证据：案例、对比、过程、售后|测评/对比/案例/FAQ|
  |热|“为什么要现在买？”|交易结构：时限、锚点、套餐、承诺|限时活动页/套餐页/结算页文案|

### 冷流量：不认识你，也不信你

  冷流量最大的阻力不是价格，是“不确定性”。用户心里只有一句话：凭什么是你？ 针对冷流量要做好的三件事：

  1. 先把用户“隐隐不爽的痛点”点出来：用户只觉得某方面别扭，你要替用户把问题讲出来，让对方心里出现一句“对，就是这个原因，这就是让我觉得生活不方便的根本原因”。
  2. 再把“为什么会这样”讲通：把因果链路解释清楚，让用户知道这不是个例，是机制导致的必然结果。不是我个人有问题，是我需要一个系统性的解决方案，去修复这个Bug。
  3. 最后把选择题摆在桌面上：把常见解法按“风险 / 成本 / 适用人群”排个序，让用户明白为什么选择你更稳、更合理。

  这个层面缺的不是优惠力度，缺的是一套“让我相信自己确实需要解决某个问题”的解释。

  ### 暖流量：认识你，但还没下决心

    暖流量的人，基本都已经被你或其他品牌都说服了“这个东西可能有用”，对这类的解决方案（或你这个品牌）已经产生了印象。
    
    在这个阶段用户不下单转化，其实卡住的不是信息量，而是三个字： 怕买错。
    
    所以这时候盲目推功能、堆参数信息的效益并不高。要做的是让用户打消心里的担忧：“我买了之后会不会踩坑？”
    
    你需要给的是能让他们“敢下决定”的背书和承诺：

      1. 案例：谁用过？用了多久？前后差了什么？最好有同类人群/同类场景。
      2. 过程透明：交付步骤是什么？要多久？每一步你做什么、需要用户做什么？出现问题怎么处理？
      3. 风险可逆转：退款/换货/试用/不满意怎么退都尽可能的要阐述清楚。

    暖流量要的不是更多信息，而是提供“我选你不会错”的确定性。
  
### 热流量：已经在比较，准备买

  热流量的人，大概率已经认可你的产品，也基本知道要买什么。真正卡住他们下决定的是：“我知道要买，但我为什么要现在买？”
  
  所以到这一环节重心不是再讲道理。是把“拖延的空间”压缩掉，把“决策的理由”摆到桌面上。
  
  在这个阶段要做的是优化交易结构：

  1. 限时窗口（给一个必须今天做决定的理由）：
       例：今晚 24:00 截止；前 100 单有效
  
  2. 价格锚点（让用户感觉自己在占便宜，而不是在花钱）：
     例：原价 1999，现在 1299；单买 A 要 799，买套装等于 A + B + 服务打包 999。
  
  4. 套餐清晰（把选择题做成一眼能选的三档）：
     例：入门版适合谁，标准版解决什么，进阶版多给什么。例：每档只说 3 个核心差异：交付内容、交付周期、支持力度。
  
  5.售后承诺（把“买错”变成“可控的试错”）：
    例：7 天无理由退款；退货包邮费。
    
  此阶段的流量不需要你传达很多信息，它只需要一个利益点，逼着用户“现在做决定”。

## 平台不是渠道，平台是筛选器：它决定你见到哪类人

  很多投放方式最常见的错误是：同一套素材、同一套话术，到处复制。
  
  但平台机制不同，筛出来的人也不同：推
    
    荐系统更擅长放大情绪与共鸣 → 更偏注意力流量
    
    搜索系统更擅长匹配关键词与需求 → 更偏意图流量


  所以你才会看到：
    
    推荐平台数据很好，但落地页就是不转化
    意图平台点击次数并不多，但 ROI 稳得吓人
    没有哪一个平台是负责成交指标的，平台只负责把某一类人筛给你。你得自己决定怎么用好不同平台的流量。
    
  大多数情况不是“投不动”，是你在“用错误场景在错误的阶段说错误的话”，而导致你不知道用户在哪个阶段，而用户也不知道你现在推给他的信息是要做什么。

## 最致命的误区：用错指标体系，好流量当垃圾，把垃圾当好流量

  增长团队最常见的自杀方式，就是“指标错配”。

### 注意力流量别用成交指标审判

  注意力流量更该看这些信号：
    
    1. 有效停留
    2. 收藏/转发（可复触达）
    3. 站内检索/品牌词增长
    4. 二次触达成本下降
    
  注意力流量的价值不在当下的 ROI，而是在一定程度上降低后续漏成的意图流量成本。

### 意图流量也别用曝光指标审判

  意图流量更认结果：CVRCACLTV/CAC复购
  
  意图流量只有一个标准：能不能成交最终的转化。

## 别再问“我要流量”，先问“我要哪一种流量”

  到最后你会发现，很多项目不是输在素材、也不是输在投放技巧。是从第一天开始就吸引错了人。
  
  开预算之前，先把“我要什么人”说清楚：我要路过的人，还是来找答案的人。

  下次你要做内容、做投放、做增长，可以按下面这张“自查清单”过一遍：
  
    1.我现在要的是注意力流量还是意图流量？
      1）如果是 注意力流量 ：我希望用户看完之后做什么？关注、收藏、进站、加私域？
      2）.如果是 意图流量：我希望用户看完之后做什么？对比、咨询、加购、下单？
      3）.检验标准：你能不能写出一句话的目标动作，而不是“先跑跑看”。

    2.我现在拉的是冷、暖还是热流量？
      1)冷：对方不认识我。我今天的任务是“把问题讲出来”，让用户形成印象，不是成交。
      2)暖：对方认识我但怕买错。我今天必须拿出：案例、对比、过程、售后里至少两项，让用户信任我。
      3)热：对方准备买但在拖。我今天必须给出：限时窗口、价格锚点、套餐选择、售后承诺的清晰规则，让用户缩短决策时间。
      4)检验标准：你能不能识别出“用户现在卡在哪一句话”。

    3. 我用的指标体系匹配得上这类流量吗？
      1)注意力流量：别盯 ROAS。先看有效停留、收藏/转发、品牌词/站内检索、再营销成本是否下降。
      2)意图流量：别盯曝光。要看 CVR、CAC、LTV/CAC、复购。
      3)检验标准：你看的指标，能不能直接指导“下一步动作”，而不是让你更焦虑。

  如果这三条你都能答清楚，你会发现投放突然变简单了：
  
  你不是再“跑跑看”，而是在按阶段推进。
  
  你也不再被报表牵着走，而是在用指标指导动作。
